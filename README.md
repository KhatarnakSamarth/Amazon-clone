# 🛒 Amazon Clone

A fully responsive **Amazon Clone** built with **React**, **TailwindCSS**, and **React Router**. This project replicates the core shopping experience of Amazon, including product listings, navigation, and a modern UI.

## 🚀 Features
- **React** for building reusable UI components  
- **TailwindCSS** for utility-first styling  
- **React Router** for seamless navigation between pages  
- Responsive design for desktop and mobile  

## 📦 Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/KhatarnakSamarth/Amazon-clone.git
cd Amazon-clone
npm install
```

## ▶️ Usage

Start the development server:

```bash
npm run dev
```

Open `http://localhost:5173` in your browser to view the app.

## 🛠 Project Structure

```bash
amazon-clone/
├── src/
│   ├── components/   # Reusable UI components
│   ├── pages/        # Page-level components (Home, Product, Cart)
│   ├── routes/       # React Router setup
│   ├── App.jsx       # Main app entry
│   └── index.css     # TailwindCSS styles
└── package.json
```

## 🎨 Styling

This project uses **TailwindCSS** for styling. Utility classes make it easy to build responsive layouts quickly.
